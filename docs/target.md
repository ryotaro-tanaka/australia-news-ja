# 🇦🇺 オーストラリア在住日本人が本当に知りたいニュースの核

## 1. 生活の安定に直結する制度・ルールの変化
**「自分の生活がどう変わるのか」**に直結する情報。

- ビザ制度（WH・学生・PR・雇用主スポンサー）
- 労働法（最低賃金、カジュアル雇用、税制）
- 医療制度（Medicare、薬不足、保険料）
- 住宅政策（家賃、補助金、住宅市場）

**→ 生活の基盤を揺らすニュースは最優先で知りたい。**

---

## 2. お金に関わる動き（収入・支出・将来）
**「生活費・収入・貯金に影響するか」**が重要。

- 金利（RBA）
- 物価・インフレ
- 電気代・ガス代の値上げ
- 雇用市場（求人動向、賃金）
- Superannuation（年金）

**→ Businessカテゴリの大半がここに該当。  
→ 日本語で読みたいニーズが非常に強い。**

---

## 3. 移民・ビザ政策の方向性（将来の不安を減らす）
**「自分の滞在・キャリアの未来に影響するか」**が最大の関心。

- 移民政策の改定
- 雇用主スポンサー制度の変更
- PR（永住権）ルールの緩和・厳格化
- 留学生政策の変更

**→ Politicsカテゴリの中でも特に重要。**

---

## 4. 安全・災害・事件（身を守るための情報）
**「今どこで何が起きているのか」**を知りたい。

- 事件・事故（治安）
- 山火事・洪水・サイクロン
- 緊急警報（Emergency Alert）
- 交通障害・ストライキ

**→ Just In の中でも生活に影響するものだけ抽出すると価値が高い。**
